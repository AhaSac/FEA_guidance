# 批处理运行

## 1. 概述

批处理是指通过命令行运行脚本，不打开软件 GUI 界面。通过在 Python 脚本中使用循环等方式，可实现有限元程序的自动化批量运行。

## 2. 批处理运行命令

### 2.1 Abaqus

```bash
abaqus cae noGUI=xxx.py
```

### 2.2 ANSYS

```bash
"C:\Program Files\ANSYS Inc\v221\commonfiles\launcherQT\source\..\..\..\ansys\bin\winx64\MAPDL.EXE" -lch -p ansys -smp -np 10 -dir 'path' -j ex00 -i pt -o po -b -l en-us -s read
```

## 3. 注意事项

### 3.1 Abaqus

- 头文件设置编码类型，避免注释报错：

```python
# -- coding: mbcs --
# -- coding: utf-8 --
```

- 不同代码块之间空两行。
- 可以采用 Abaqus 终端检测运行，也可以通过 PDE 检测运行。
- PDE 需选用内核运行。
- 面的选择需要注意编号，不要盲目复制 `.rpy` 文件，网上有较多可靠写法。
- `session.viewports` 及部分界面约束命令通常无实际作用，可删掉以避免异常报错。

### 3.2 ANSYS

- 由于 APDL 运行方式中错误较多、部分问题排查困难，建议加入跳过报错相关命令：

```apdl
/uis,msgpop,3
keyw,pr_sgvof,1
/nerr,0,,,,0
```

- 每个部分开始前运行一次清空选中，可避免很多报错：

```apdl
ALLSEL
FINISH
```

- 部分 APDL 命令可能无法直接运行（如部分后处理导出），可写入 `.mac` 文件再调用。

## 4. Abaqus 算例

### 4.1 外压球壳内加筋屈曲分析

基本参数：

- 壳单元网格：`S4R`
- 材料：铝合金（杨氏模量 73000，泊松比 0.33，密度 2.79e-9 `g/mm³`）
- 网格划分：以四边形为主，自由中轴线算法
- 载荷：外压 0.6 MPa
- 边界条件：固定顶点
