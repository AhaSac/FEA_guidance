# Abaqus 静力学分析

## 1. 参数化建模

- 采用 Python 语言，针对内核进行开发，进行批量计算。
- 分为 `mdb` 和 `odb` 两大部分，分别对应前处理建模与后处理数据整理。
- 代码可以通过 Abaqus 自带的 `PDE` 模块进行调试。
- `PDE` 模块中可加入断点，并结合 GUI 界面观察代码运行情况。
- 可搭配 `PythonReader`，或直接在工作目录内查找 `.rpy` 文件，提取 GUI 界面对应操作的 Python 代码。

### 1.1 记录方式设置

将选中区域由坐标点改为编号：

```python
session.journalOptions.setValues(replayGeometry=COORDINATE,recoverGeometry=COORDINATE)
```

## 2. 分析

### 2.1 静力（通用）

- 设置时间长度和最大增量步。
- 若发生屈曲响应，则增量步会逐渐减小，时间步逐渐收敛。
- 极限载荷计算：`极限载荷 = 时间步 / 时间长度 * 载荷值`

### 2.2 静力（Riks）

- 控制增量步，观察后屈曲响应。

### 2.3 屈曲

- 特征值屈曲分析可得到屈曲特征值。
- 一般取前三阶，并选取较理想数据代入计算（通常为一阶特征值）。
- 屈曲载荷计算：`屈曲载荷 = 载荷值 * 特征值`
